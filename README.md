
<h1 align="left">Описание</h1>

<p align="left">Меня зовут **Николай** и я аналитик данных из Казахстана, г. Алматы с коммерческим опытом более 1 года. Принес огромную ценность для международных компаний благодаря проактивному исследовательскому анализу. Готов принести выгоду и вам!</p>

2. Финансы (инвестиции, банковские операции, управленческая отчетность)
3. Продажи (FMCG)
